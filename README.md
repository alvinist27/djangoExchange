# djangoExchange

## Описание
Конвертер валют — инструмент, который позволит рассчитать соотношения актуальных курсов денежных средств всего мира на сегодня.
Конвертация производится по курсу согласно exchangerate.host API.

## Технологии
* Django
* HTML
* CSS
* Bootstrap
* Django Crispy Forms
* Docker
* API

## Инструменты Django
* Формы
* Тестирование
* Интернационализация
* Кэширование
